# Criação do Cardápio em Python

Este projeto implementa a **criação de um cardápio em Python** seguindo o **diagrama conceitual** definido previamente.

---

## Observação Importante
Algumas classes, como **`Pagamento`**, aparecem no diagrama apenas como **conceitos**.  
Ou seja, **não representam objetos funcionalmente implementados no código**, mas sim elementos teóricos do modelo.

Isso serve para manter a coerência entre o projeto e o diagrama conceitual, mesmo que nem todos os elementos sejam utilizados diretamente na aplicação.
